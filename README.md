# A02 Ping Pong ML Collaboration

## Description
This assignment demonstrates a collaborative machine learning workflow using GitHub branching, pull requests, and peer reviews. The project builds an end-to-end regression pipeline using the California Housing dataset from scikit-learn and an `MLPRegressor` neural network model with customized hyperparameters and early stopping. The workflow follows a ping-pong collaboration strategy where both team members contribute through separate branches and reviewed pull requests.

## Objectives
- Practice collaborative software development using GitHub
- Implement a machine learning regression pipeline
- Train and evaluate a neural network regression model
- Generate actual vs. predicted visualizations for training and testing datasets
- Apply GitHub best practices including branch protection and pull request reviews
